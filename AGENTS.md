# Agents

This project uses AI agents in the development workflow. All agents should follow the guidelines in [CLAUDE.md](./CLAUDE.md).

## Invoking the agent

- **Issues & PR comments**: Mention `@claude` in a comment to trigger the agent.
- **New issues**: Include `@claude` in the issue body or title to trigger on creation.
- **PR reviews**: Include `@claude` in the review body.

## What the agent can do

- Triage and analyze issues
- Implement features and fix bugs
- Create pull requests
- Respond to review feedback and push fixes
- Run scheduled checks (Lighthouse, etc.)
